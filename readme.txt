Ein kleines Plugin welches Shortcodes zum einbinden von wikifolio Charts bereitstellt.

== Description ==
Mit diesem Plugin kann man ganz ein­fach wiki­fo­lios in eine Word­Press Web­seite inte­grie­ren. Egal, ob in einem Blog­post, in der Sei­ten­leiste oder dem Fuß­be­reich der Seite. Mit ver­schie­de­nen Short­codes kann der Graph fast über­all ange­zeigt wer­den. Zudem ist es nicht nur mög­lich dein eigenes wiki­fo­lio dar­zu­stel­len, son­dern auch jedes andere wiki­fo­lio auf das bspw. in einem Arti­kel näher ein­ge­gan­gen wer­den soll. Es ist also nicht unbe­dingt nur für wiki­fo­lio Tra­der geeig­net, son­dern auch für andere Web­sei­ten­be­trei­ber, die über wiki­fo­lios berichten.

[Mehr Informationen zum Plugin](http://TrendFollowing.de/wikifolio-chart-plugin)

== Installation ==
Nach­dem das Plugin instal­liert wurde, muss es nur noch akti­viert wer­den. Dann ste­hen die Short­codes auch schon zur Ver­fü­gung und kön­nen in Posts, Wid­gets und Sei­ten ver­wen­det wer­den.

Der Short­code für das TF-​wikifolio-​Chart Plugin lau­tet ent­we­der `[tfwc]` oder `[tf-wikifolio-chart]`. Je nach­dem, ob eine kür­zere oder ein­deu­ti­gere Vari­ante gewünscht ist.

Um ein Wid­get mit einem Chart zu nut­zen, kann man ein Text-​Widget anle­gen und in die­sem den Short­code nut­zen.

In Posts und Sei­ten kann der Short­code direkt ver­wen­det wer­den.

Um einen genauen Über­blick über die Mög­lich­kei­ten des Short­codes zu bekom­men, bit­ten wir dich die [Bei­spiel Gra­phen anzu­se­hen](http://www.trendfollowing.de/wikifolio-chart-plugin/ "Beispiel Graphen auf TrendFollowing.de"). Dort kann man gut die Ver­schie­de­nen Mög­lich­kei­ten erken­nen.


= Shortcodes =
* `[tfwc name-display="TF Value Momentum - 2016-02-15 bis 2016-03-10" name="351f05dc-e8c9-4d46-866e-a368922809cd" date-from="2016-02-15" date-to="2016-03-10"]`
* `[tfwc name-display="TF Value Momentum - letzte 30 Tage" name="351f05dc-e8c9-4d46-866e-a368922809cd" last-days="30"]`
* `[tfwc name-display="DACH-Trading&Invest - 2016-01-01 bis heute" name="dbc8abe8-d2ce-4c56-8ea7-5c07b08ae046" date-from="2016-01-01"]`
* `[tfwc name-display="DACH-Trading&Invest - 2016-03-29 und die letzten 10 Tage davor" name="dbc8abe8-d2ce-4c56-8ea7-5c07b08ae046" date-to="2016-03-29" last-days="10"]`
